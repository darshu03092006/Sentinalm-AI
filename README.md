# Sentinel AI

Sentinel AI is an AI-powered surveillance system designed to detect suspicious activities in real time using computer vision and artificial intelligence.

## Features

- Real-time CCTV monitoring
- Suspicious activity detection
- AI-based object detection
- Instant alert system
- Live camera feed analysis

## Technologies Used

- Python
- OpenCV
- YOLOv8
- FastAPI
- NumPy

## Project Structure

```bash
Sentinel-AI/
│
├── main.py
├── detect.py
├── alert.py
├── requirements.txt
├── README.md
└── screenshots/
```

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Sentinel-AI.git
```

### 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Run the Project

```bash
python main.py
```
